<h1 style= "text-align:center">Shopping RESTful API</h1>

This is a simple node shopping rest api that has three (3) resources; ORDER, PRODUCT and USER.

---

## Technologies used:

* Nodejs
* Expressjs
* Bcrypt
* Nodemon
* Mongoose ODM
* MONGODB
* Jsonwebtoken
